Web Development Course Syllabus

Welcome to the Web Development Course. Below is the syllabus for the course:

  • Unit 1: Introduction to HTML and Github
  • Unit 2: CSS and Its Frameworks
  • Unit 3: JavaScript
  • Unit 4: ECMA Script 6
  • Unit 5: Introduction to React JS

Unit 1: HTML and GitHub

Introduction: HTML Tags, Grouping Using Division tag and Span, Lists, Images, Hyperlink, Table, Form, XHTML Multimedia. Web Designing: Introduction to Web Designing, Web Designing Strategies, Web Designing Issues, designing using Figma.
GitHub Introduction: Create a New Local Git Repository, Stage & Commit Files, push to a Remote Repository, Pull from a Remote Repository, Clone (Download) a Remote Repository, Version control. Hands-on: Static website development using HTML.

Related Youtube Video Content

Unit 2: CSS and Its Frameworks

Introduction: Syntax, Types of CSS, Margins, Padding, Fonts, Text, Overflow, Float, Navigation Bar, Image Gallery, Rounded Corners, Multiple Backgrounds, Gradients, Shadow Effects, Transitions, Animations, Multiple Style Sheets, Selectors, Box Model.
Bootstrap: Introduction to Bootstrap, Bootstrap Grid layout, Bootstrap Components, Responsive pages.
Tailwind CSS: Introduction to Tailwind, Classes of Tailwind, Flex and Grid layout, Responsive pages.
Hands-on: Website designing using CSS.

Related Youtube Video Content

Unit 3: JavaScript

Introduction: Data types, operators, variables, substring, array, objects, conditional statements, loops, validation, Client-Side scripting versus Server-side scripting.
HTML DOM and JavaScript: Finding HTML Elements, Manipulating DOM Elements with JavaScript. Creating Functions in JavaScript, UI Events and event handling, Returning Data from Functions, working with Conditions, looping in JavaScript, Screen output and keyboard input, Block Scope Variables, working with Objects, Creating Object using Object Literals.
Hands-on: Interactive web designing using JavaScript..

Related Youtube Video Content

Unit 4: ECMA Script 6

Introduction: let, var, const, Default function parameters, Rest parameters, Spread parameters, Object literal syntax extensions, Template literals, De-structuring, ES6 Modules, Arrow functions, Async/Awailt, Promises, Callback methods.
Hands-on: Asynchronous Application Development.

Related Youtube Video Content

Unit 5: Introduction to React JS

React Fundamentals: Introduction to React, Running first React App using VS Code, Rendering JSX & JSX Expression, Rendering Multiple Elements, Rendering List, Conditonal Rendering, Functional Components, Props, Passing Props to Components, Bootstrap in React, Dynamic Class Rendering, Dynamic Expression Rendering, Event Handling, Component Composition, Passing Events to Components, Components Styling, Working with Forms.
Hands-on: Dynamic Website using React.
